# Frontend Development Guidelines — N/A

> **This project has no frontend.**
>
> `ship/` is a Python data-processing pipeline. There is no UI, no web
> server, no browser-facing code. The only "visualization" outputs are
> static figures (`figures/*.pdf`, `figures/*.png`) generated by
> `code/plot_*.py` scripts using matplotlib / cartopy — they are
> documented under the backend spec.
>
> The `.trellis/spec/frontend/` directory exists because `trellis init`
> created it as a fullstack scaffold; the placeholder files inside it are
> intentionally left untouched and should be ignored by AI sub-agents.
>
> If a future component of this project ever grows a real frontend (web
> dashboard, inspector tool, etc.), delete this notice and populate the
> sibling spec files with real conventions.
